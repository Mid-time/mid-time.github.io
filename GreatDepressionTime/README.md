# 大萧条时代 (Great Depression Time)

一个基于1929年经济大萧条背景的股票生存游戏。

## 游戏概述

在1929年股市崩盘后的经济大萧条中，你作为一名普通投资者，需要管理你的资金、健康和生活，在艰难的股市环境中生存下去。

## 游戏特点

### 核心玩法
- **股票交易**: 买卖股票，把握市场时机
- **生存管理**: 管理健康、饥饿和现金
- **时间推进**: 白天/黑夜交替，时间影响一切
- **随机事件**: 各种突发事件影响游戏进程

### 游戏模式
- **简单模式**: 28天生存目标，无房租压力
- **正常模式**: 336天生存目标，每月支付房租

### 特色系统
- **股票系统**: 动态生成的股票，有崩盘和新增机制
- **健康系统**: 健康值归零游戏结束
- **事件系统**: 个人、市场、社会三类随机事件
- **季节系统**: 季节影响股票价格
- **难度系统**: 不同难度不同挑战

## 运行方式

1. 直接打开 `index.html` 在浏览器中运行
2. 或部署到任意静态网站服务器

## 游戏控制

### 基本操作
- **导航栏**: 切换股票市场和生存指南界面
- **时间推进**: 点击右上角按钮推进时间
- **股票交易**: 在股票界面选择股票，输入数量，点击买入/卖出
- **生存管理**: 使用左下角的进食和治疗按钮

### 移动端
- 建议使用横屏模式
- 触摸操作完全支持

## 游戏规则

### 胜利条件
- 简单模式: 存活28天
- 正常模式: 存活336天（1年）

### 失败条件
- 健康值归零

### 核心机制
1. **时间系统**: 每点击一次时间按钮推进半天
2. **健康系统**: 随时间自然消耗，需通过进食和治疗维持
3. **股票系统**: 股票价格随机波动，可能崩盘和新增
4. **事件系统**: 随机事件影响游戏进程
5. **房租系统**: 正常模式每月需支付房租
